# Artemi — Luxury Handicrafts

Official website for **Artemi**, a premium handicrafts studio specialising in white-labeling for luxury brands.

## Pages

- **Homepage** (`index.html`) — Hero, About, Services, Featured Products, CTA
- **Products** (`products.html`) — Full product collection grid
- **Contact** (`contact.html`) — Enquiry form for brand partnerships

## Tech

- Pure HTML + CSS (no frameworks)
- Google Fonts — Playfair Display + Jost
- Hosted on [Vercel](https://vercel.com)
- Domain: [artemi.in](https://artemi.in)

## Deployment

Connected to Vercel via GitHub. Any push to `main` branch auto-deploys to production.
